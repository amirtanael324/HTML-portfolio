HTML & CSS Guide

This contains the HTML commands that we have learned throughout the class, and CSS for adittional Knowledge.


HTML Commands

h1 to h6

h1 to h6 are used to create headings in HTML. h1 is the largest and most important heading, while h6 is the smallest and least important. They are used to structure the content of a webpage and make it easier to read.

p

The p tag is used to create paragraphs in HTML. It is a block-level element that is used to group together related sentences and provide structure to the content of a webpage.

ul and ol

ul and ol are used to create lists in HTML. ul stands for unordered list, which creates a bulleted list, while ol stands for ordered list, which creates a numbered list. They are used to organize information and make it easier to read.

li

The li tag is used to create list items in HTML. It is used within ul and ol tags to define each item in the list. Each li element represents a single item in the list.

img

The img tag is used to embed images in HTML. It requires the src attribute to specify the source of the image, and the alt attribute to provide alternative text for accessibility. The width and height attributes can also be used to specify the dimensions of the image.

footer

This is a simple text added at the very bottom of the page.

main

The main tag is used to define the main content of a webpage. It is a block-level element that is used to group together the primary content of a webpage, such as articles, sections, and other relevant information. The main tag helps to improve the structure and accessibility of a webpage.

article

The article tag is used to define a self-contained piece of content in HTML. It is a block-level element that is used to group together related information, such as a blog post, news article, or product description. The article tag helps to improve the structure and organization of a webpage.

section

The section tag is used to define a section of content in HTML. It is a block-level element that is used to group together related information, such as a chapter in a book, a section of a webpage, or a group of related articles. The section tag helps to improve the structure and organization of a webpage.

nav

The nav tag is used to define a navigation menu in HTML. It is a block-level element that is used to group together links to other pages or sections of a website. The nav tag helps to improve the structure and accessibility of a webpage by providing a clear and organized way for users to navigate through the site.

CSS Commands

color

The color property in CSS is used to specify the color of text. It can be defined using various formats, such as named colors (e.g., red, blue), hexadecimal values (e.g., #ff0000), RGB values (e.g., rgb(255, 0, 0)), or HSL values (e.g., hsl(0, 100%, 50%)). The color property helps to enhance the visual appeal and readability of a webpage.

background-color

The background-color property in CSS is used to specify the background color of an element. It can be defined using various formats, such as named colors (e.g., yellow, green), hexadecimal values (e.g., #ffff00), RGB values (e.g., rgb(255, 255, 0)), or HSL values (e.g., hsl(60, 100%, 50%)). The background-color property helps to enhance the visual appeal and readability of a webpage by providing a contrasting background for text and other elements.

font-size

The font-size property in CSS is used to specify the size of text. It can be defined using various units, such as pixels (e.g., 16px), ems (e.g., 1em), rems (e.g., 1rem), percentages (e.g., 100%), or keywords (e.g., small, medium, large). The font-size property helps to enhance the readability and visual hierarchy of a webpage by allowing designers to control the size of text elements.

font-family

The font-family property in CSS is used to specify the typeface of text. It can be defined using a list of font names, where the browser will use the first available font in the list. For example, font-family: "Arial", sans-serif; will use Arial if it is available, and if not, it will fall back to a generic sans-serif font. The font-family property helps to enhance the visual appeal and readability of a webpage by allowing designers to choose appropriate fonts for their content.

border

The border property in CSS is used to specify the border around an element. It can be defined using various values, such as border-width (e.g., 1px), border-style (e.g., solid, dashed, dotted), and border-color (e.g., black, #000000). The border property helps to enhance the visual appeal and structure of a webpage by providing a clear separation between elements and adding emphasis to certain areas.

margin

The margin property in CSS is used to specify the space around an element. It can be defined using various units, such as pixels (e.g., 10px), ems (e.g., 1em), rems (e.g., 1rem), or percentages (e.g., 10%). The margin property helps to enhance the layout and spacing of a webpage by providing separation between elements and improving the overall visual appeal.

padding

The padding property in CSS is used to specify the space between the content of an element and its border. It can be defined using various units, such as pixels (e.g., 10px), ems (e.g., 1em), rems (e.g., 1rem), or percentages (e.g., 10%). The padding property helps to enhance the layout and spacing of a webpage by providing separation between the content and the border of an element, improving readability and visual appeal.
